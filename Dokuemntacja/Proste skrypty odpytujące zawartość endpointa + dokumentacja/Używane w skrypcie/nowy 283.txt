# DataForSEO Historical Keyword Data API - Kompletna Dokumentacja

## ENDPOINT INFO
| Parameter | Value |
|-----------|-------|
| **URL** | https://api.dataforseo.com/v3/dataforseo_labs/google/historical_keyword_data/live |
| **Method** | POST |
| **Format** | JSON (UTF-8) |
| **Rate Limit** | 2000 calls/minute |
| **Concurrent Limit** | 30 simultaneous requests |
| **Max Keywords** | 700 per request |
| **Keyword Limits** | Max 80 chars, max 10 words per phrase |

---

## REQUEST PARAMETERS
| Pole | Typ | Wymagane | Default | Opis |
|------|-----|----------|---------|------|
| keywords | array | **TAK** | - | Lista słów kluczowych (max 700). Automatycznie konwertowane na lowercase |
| location_name | string | TAK* | - | Pełna nazwa lokalizacji (wymagane jeśli brak location_code) |
| location_code | integer | TAK* | - | Kod lokalizacji (wymagane jeśli brak location_name) |
| language_name | string | TAK** | - | Pełna nazwa języka (wymagane jeśli brak language_code) |
| language_code | string | TAK** | - | Kod języka (wymagane jeśli brak language_name) |
| tag | string | NIE | null | Identyfikator zadania (max 255 znaków) |

**\*Wymagane**: location_name LUB location_code  
**\*\*Wymagane**: language_name LUB language_code

---

## RESPONSE STRUCTURE

### ROOT LEVEL
| Pole | Typ | Opis |
|------|-----|------|
| version | string | Aktualna wersja API |
| status_code | integer | Generalny kod statusu |
| status_message | string | Generalna wiadomość informacyjna |
| time | string | Czas wykonania w sekundach |
| cost | float | Całkowity koszt zadań w USD |
| tasks_count | integer | Liczba zadań w tablicy tasks |
| tasks_error | integer | Liczba zadań zwróconych z błędem |
| tasks | array | Tablica zadań |

---

### TASKS STRUCTURE
```
tasks[]
├── id (string) - Unikalny identyfikator zadania (UUID)
├── status_code (integer) - Kod statusu zadania (10000-60000)
├── status_message (string) - Wiadomość informacyjna zadania
├── time (string) - Czas wykonania w sekundach
├── cost (float) - Koszt zadania w USD
├── result_count (integer) - Liczba elementów w result array
├── path (array) - Ścieżka URL
├── data (object) - Parametry z POST request
└── result (array) - Tablica wyników
```

---

### RESULT STRUCTURE
```
result[]
├── se_type (string) - Typ wyszukiwarki
├── location_code (integer) - Kod lokalizacji z POST
├── language_code (string) - Kod języka z POST
├── items_count (integer) - Liczba wyników w items array
└── items (array) - Słowa kluczowe i powiązane dane
```

---

### ITEMS STRUCTURE
```
items[]
├── se_type (string) - Typ wyszukiwarki
├── keyword (string) - Słowo kluczowe (dekodowane z %##)
├── location_code (integer/null) - Kod lokalizacji
├── language_code (string) - Kod języka
└── history (array) - Dane historyczne dla słowa kluczowego
```

---

### HISTORY STRUCTURE
```
history[]
├── year (integer) - Rok
├── month (integer) - Miesiąc
└── keyword_info (object) - Dane historyczne słowa kluczowego
```

### KEYWORD_INFO STRUCTURE
| Pole | Typ | Opis |
|------|-----|------|
| se_type | string | Typ wyszukiwarki |
| last_updated_time | string | Data i czas aktualizacji (UTC: "yyyy-mm-dd hh-mm-ss +00:00") |
| competition | float | Konkurencja (0-1, bazowane na Google Ads) |
| competition_level | string/null | Poziom konkurencji: LOW/MEDIUM/HIGH |
| cpc | float | Średni koszt za kliknięcie (USD) |
| search_volume | integer | Średni miesięczny wolumen wyszukiwań |
| low_top_of_page_bid | float/null | Minimalna stawka za wyświetlenie na górze strony |
| high_top_of_page_bid | float/null | Maksymalna stawka za wyświetlenie na górze strony |
| categories | array | Kategorie produktów i usług |
| monthly_searches | array | Miesięczne wyszukiwania |
| search_volume_trend | object | Trendy zmian wolumenu wyszukiwań |

#### KEYWORD_INFO TREE VIEW
```
keyword_info
├── se_type (string) - Typ wyszukiwarki
├── last_updated_time (string) - Ostatnia aktualizacja danych
├── competition (float) - Konkurencja 0-1
├── competition_level (string/null) - LOW/MEDIUM/HIGH
├── cpc (float) - Koszt za kliknięcie USD
├── search_volume (integer) - Miesięczny wolumen
├── low_top_of_page_bid (float/null) - Min stawka top page
├── high_top_of_page_bid (float/null) - Max stawka top page
├── categories (array) - Kody kategorii produktów
├── monthly_searches (array) - Dane miesięczne za 12 miesięcy
│   ├── year (integer) - Rok
│   ├── month (integer) - Miesiąc
│   └── search_volume (integer) - Wolumen w danym miesiącu
└── search_volume_trend (object) - Zmiany procentowe wolumenu
    ├── monthly (integer/null) - Zmiana vs poprzedni miesiąc (%)
    ├── quarterly (integer/null) - Zmiana vs poprzedni kwartał (%)
    └── yearly (integer/null) - Zmiana vs poprzedni rok (%)
```

## DOSTĘPNE DANE HISTORYCZNE
- **Zakres czasowy**: Od sierpnia 2021 (w zależności od kombinacji keyword/location/language)
- **Aktualizacja**: Raz w miesiącu po aktualizacji źródła danych
- **Źródło danych**: Google Ads API
- **Uwaga**: Google czasami przepisuje wartości wolumenu dla danego miesiąca

## UWAGI TECHNICZNE
1. **Lowercase conversion**: Wszystkie keywords są automatycznie konwertowane na małe litery
2. **Brak danych**: Jeśli keyword nie występuje w wynikach, baza danych go nie zawiera
3. **Brak opłat**: Nie płacisz za keywords pominięte w wynikach
4. **Null values**: Niektóre pola mogą być null jeśli dane nie są dostępne
5. **Categories**: Można pobrać pełną listę kategorii z osobnego endpointu






import os
import logging
from fastapi import APIRouter, HTTPException
from pydantic import BaseModel
from dotenv import load_dotenv
from dataforseo_client import configuration as dfs_config, api_client as dfs_api_provider
from dataforseo_client.api.dataforseo_labs_api import DataforseoLabsApi
from dataforseo_client.models.dataforseo_labs_google_historical_keyword_data_live_request_info import (
    DataforseoLabsGoogleHistoricalKeywordDataLiveRequestInfo,
)

load_dotenv()
router = APIRouter()

# Logger
logger = logging.getLogger("dfs_labs_historical")
logger.setLevel(logging.DEBUG)
handler = logging.StreamHandler()
formatter = logging.Formatter('%(asctime)s | %(levelname)s | %(message)s')
handler.setFormatter(formatter)
logger.addHandler(handler)

DFS_LOGIN = os.getenv("DATAFORSEO_LOGIN")
DFS_PASSWORD = os.getenv("DATAFORSEO_PASSWORD")

# Model danych wejściowych
class HistoricalKeywordInput(BaseModel):
    keywords: list[str]
    location_code: int = 2616
    language_code: str = "pl"

@router.post("/labs/historical-keyword-data")
async def get_historical_keyword_data(data: HistoricalKeywordInput):
    if not DFS_LOGIN or not DFS_PASSWORD:
        logger.error("❌ Brak danych logowania do API")
        raise HTTPException(status_code=500, detail="Brak danych logowania do API DataForSEO.")

    logger.info(f"➡️ Zapytanie dla słów kluczowych: {data.keywords}")

    config = dfs_config.Configuration(username=DFS_LOGIN, password=DFS_PASSWORD)
    request_data = [
        DataforseoLabsGoogleHistoricalKeywordDataLiveRequestInfo(
            keywords=data.keywords,
            location_code=data.location_code,
            language_code=data.language_code
        )
    ]

    try:
        with dfs_api_provider.ApiClient(config) as api_client:
            api_instance = DataforseoLabsApi(api_client)

            logger.debug("➡️ Wysyłanie żądania do DataForSEO Labs API (Historical Keyword Data)...")
            api_response = api_instance.google_historical_keyword_data_live(request_data)

            task = api_response.tasks[0]
            if not task.result:
                logger.warning("⚠️ Brak wyników dla podanych słów kluczowych.")
                raise HTTPException(status_code=404, detail="Brak danych dla podanych słów kluczowych.")

            logger.info("✅ Pobrano dane z DFS Labs API (Historical Keyword Data).")
            return task.result[0].to_dict()

    except Exception as e:
        logger.exception("❌ Błąd podczas pobierania danych z DFS Labs API")
        raise HTTPException(status_code=500, detail=f"Błąd serwera:\n{str(e)}")
